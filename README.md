# 🔲 ESP32-017: Ball Control using OLED and MPU6500 🔲

Control a virtual ball on an OLED display using the ESP32 and MPU6500 motion sensor. The MPU6500 detects tilt and movement, while the ESP32 updates the ball's position in real time, demonstrating motion sensing, I2C communication, and interactive graphics on a compact OLED screen.

## 🛠️ Components Required

- [ESP32 Development Board](https://robocraze.com/products/nodemcu-32-wifi-bluetooth-esp32-development-board30-pin)
- [MPU6500 Module](https://robocraze.com/products/mpu-9250-module)
- [1.3" OLED Display](https://robocraze.com/products/1-3in-oled-display)
- [Jumper Wires](https://robocraze.com/products/f2m-jumper-wires-20cm-20pcs)
- [Mini Breadboard](https://robocraze.com/products/mini-breadboard)
- [USB to Micro USB Cable](https://robocraze.com/products/usb-to-micro-usb-cable)
